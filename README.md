# Campus Notification System

## Stage 1

This project fetches notifications from the provided API and displays the top 10 priority notifications.

## Features

- Fetch notifications from API
- Priority based sorting
- Latest notifications first
- Logging middlewaregit push
- Top 10 notifications display
